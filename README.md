# STM32_App_Radio_control 

This project develops an STM32 control system using a Processing Java application. Messages are transmitted from the PC to a radio transmitter (STM32_103), and from there to a radio receiver (STM32_407), where LEDs indicate operation.

В этом проекте разрабатывается система управления STM32 с помощью Processing Java приложения. Передача сообщения с ПК идёт на радиопередатчик(STM32_103), с него на радиоприёмник (STM32_407), на котором работу отображают светодиоды.
